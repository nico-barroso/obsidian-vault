# Variables en una sola línea

## Asignación múltiple

```python
name, surname, alias, age = "Brais", "Moure", "MoureDev", 35
```

## Intercambio de valores

```python
a, b = 1, 2
a, b = b, a  # Sin variable temporal
```

## Desempaquetado

```python
a, b, c = [1, 2, 3]
x, y, z = "abc"
```
